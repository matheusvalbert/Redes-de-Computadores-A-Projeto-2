# Introdução
```
Este projeto foi desenvolvido no sétimo semestre na disciplina de Redes de Computadores A.
O objetivo deste projeto foi criar um software de envio de mensagens e imagens peer-to-peer utilizando os 
protocólos TCP/IP.
Neste caso o servidor é utilizado para armazenar os IPs dos clientes e os clientes podem se comunicar
através do recebimento das informações de seu contato pelo servidor.
```
# Começando a utilizá-lo
```
Este projeto deve ser utilizado em um sistema operacional Linux.
Primeiramente após realizar o download dos arquivos deve-se utilizar o comando make
na pasta onde os arquivos se encontram.
Serão gerados dois arquivos, um nomeado "client" e um nomeado "server", deve-se executar o arquivo chamado
"server" no terminal e executar o arquivo client no terminal.
Pode-se executar o servidor em um computador e os clientes em diferentes computadores, desde que estejam na 
mesma rede interna.
```
